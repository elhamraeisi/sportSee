# SportSee

SportSee is a startup dedicated to sports coaching.

## Prerequisites

- NodeJS (version 12.18)
- npm

## Installation

### Back-end

In order to run the application, first step is to run the backend.
You can clone and run the back-end from this repo : https://github.com/OpenClassrooms-Student-Center/P9-front-end-dashboard

Then install all dependencies using : `npm install`

### Front-end

To install the front-end project, you should follow next steps :

1- Clone this repository

2- Install all dependencies using : `npm install`

## Launching the project

### Back-end

The `yarn dev` command will allow you to run the API.

### Front-end
Run `npm run start` to run the front-end server
